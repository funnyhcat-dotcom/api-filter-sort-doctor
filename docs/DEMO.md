# Demo

Run against the strong example:

```bash
node bin/api-filter-sort-doctor.js examples/good.md --min-score 90
```

Run against the weak example:

```bash
node bin/api-filter-sort-doctor.js examples/bad.md --min-score 80
```

Use JSON in bots:

```bash
node bin/api-filter-sort-doctor.js examples/good.md --json
```
